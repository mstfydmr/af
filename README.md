# AF

## Install & Upgrade
```bash
curl -sSL https://raw.githubusercontent.com/mstfydmr/af/master/install.sh | bash
```

# Usage Commands

## Overview
```bash
af
```

## Open MRs
```bash
af mr
```

## Create New MR
```bash
af mr {target-branch}
af mr master # Create a MR to `master`
af mr staging # Create a MR to `staging`
...
```
